## 🚀 Project Overview

The **IEA Chatbot Project** is a conversational AI application that simulates human-like interaction through an NLP-powered backend and an interactive frontend interface.

The system demonstrates:

- user input processing
- NLP-based response generation
- frontend–backend communication
- modular chatbot architecture design

This project was developed as part of the **Intelligent Enterprise Applications (IEA)** coursework to showcase practical implementation of AI-driven chatbot systems.

## ⚙️ Installation Guide

Clone the repository:

git clone https://github.com/AANIYAMEHRA/IEA_chatbot_project.git

Move into the project directory:

cd IEA_chatbot_project


---

## ▶️ Backend Setup

Navigate to backend folder:

cd backend

Install dependencies:

pip install -r requirements.txt

Run backend server:

python app.py


---

## 💻 Frontend Setup

Navigate to frontend folder:

cd frontend

Install dependencies:

npm install

Start frontend server:

npm start
